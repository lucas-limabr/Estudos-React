## Estudos da biblioteca React
Este repositório tem como base criar casos de uso mínimos a fim de estudar o React. Ele é focado em front-end, mas em alguns diretórios há códigos que testam alguma API pública, utilizando-se do fecth ou axios do React. Os exemplos são simples e em sua maioria não tem aplicação real, mas serve para aprender conceitos essenciais do React
